# Deploy to GitHub Pages - Instant Free Hosting

## Quick Deploy Instructions

### Step 1: Create GitHub Account
1. Go to https://github.com/signup
2. Create free account

### Step 2: Create Repository
1. Click "+" → "New repository"
2. Name: `renuka-law-associates`
3. Set to Public
4. Click "Create repository"

### Step 3: Upload Files
1. Click "uploading an existing file"
2. Drag ALL files from this package
3. Commit changes

### Step 4: Enable GitHub Pages
1. Go to Settings → Pages
2. Source: Deploy from branch "main"
3. Folder: / (root)
4. Click Save

### Your Website Will Be Live At:
```
https://YOUR-USERNAME.github.io/renuka-law-associates/
```

**Time Required**: 5-10 minutes
**Cost**: FREE forever
**SSL Certificate**: Automatic
**Custom Domain**: Supported (add in settings)

## Alternative: Use GitHub Desktop (Even Easier)

### Step 1: Download GitHub Desktop
https://desktop.github.com/

### Step 2: Create New Repository
1. File → New Repository
2. Name: renuka-law-associates
3. Choose folder with website files

### Step 3: Publish
1. Click "Publish repository"
2. Uncheck "Keep this code private"
3. Click "Publish repository"

### Step 4: Enable Pages
1. Go to repository on GitHub.com
2. Settings → Pages → Enable

**Done! Your site is live.**

## Add Custom Domain (Optional)

### If you have renukalawassociates.com:

1. In repository, create file: `CNAME`
   Content: `www.renukalawassociates.com`

2. In domain registrar (GoDaddy, Namecheap, etc.):
   Add CNAME record:
   - Type: CNAME
   - Name: www
   - Value: YOUR-USERNAME.github.io

3. Wait 24-48 hours for DNS propagation

**Your site will be at**: www.renukalawassociates.com
